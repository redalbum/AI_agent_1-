===============================================================
   AI БИЗНЕС-АССИСТЕНТ ДЛЯ 1С
   Telegram-бот для анализа данных и рекомендаций
===============================================================

БЫСТРЫЙ СТАРТ (5 минут)
---------------------------------------------------------------

ШАГ 1: УСТАНОВКА
   • Дважды кликните на INSTALL.bat
   • Если спросит права администратора — разрешите
   • Дождитесь сообщения "Установка завершена!"

ШАГ 2: ПОЛУЧИТЕ ТОКЕН TELEGRAM-БОТА
   1. Откройте Telegram на телефоне или компьютере
   2. В поиске найдите: @BotFather
   3. Напишите ему: /newbot
   4. Придумайте название (например: Мой 1С Ассистент)
   5. Придумайте username (например: my_1c_assistant_bot)
   6. BotFather пришлёт токен — скопируйте его
      Токен выглядит так: 7819543862:AAF7kLx9vZ...

ШАГ 3: УЗНАЙТЕ СВОЙ TELEGRAM ID
   1. В Telegram найдите: @userinfobot
   2. Напишите ему: /start
   3. Скопируйте число из строки "Id:"
      ID выглядит так: 123456789

ШАГ 4: ПОЛУЧИТЕ КЛЮЧ OPENROUTER (бесплатно)
   1. Откройте в браузере: https://openrouter.ai
   2. Нажмите Sign Up (регистрация через Google/GitHub)
   3. После входа откройте: Keys (в меню слева)
   4. Нажмите: Create Key
   5. Скопируйте ключ (начинается с sk-or-...)

ШАГ 5: ЗАПОЛНИТЕ НАСТРОЙКИ
   1. На рабочем столе откройте: "Nastroyki_bota"
   2. Заполните:
      • TELEGRAM_BOT_TOKEN = ваш токен от BotFather
      • TELEGRAM_ALLOWED_USERS = ваш ID от userinfobot
      • ONEC_CONNECTION_STRING = путь к базе 1С
      • PROVIDER_API_KEY = ключ от OpenRouter
   3. Сохраните файл: Ctrl+S

ШАГ 6: ЗАПУСТИТЕ БОТА
   1. На рабочем столе кликните: "Zapustit_bota"
   2. Дождитесь сообщения "Бот работает!"
   3. В Telegram откройте вашего бота
   4. Напишите: /start

ГОТОВО!


ЧТО УМЕЕТ БОТ
---------------------------------------------------------------

АНАЛИТИКА ПРОДАЖ
   • Топ продаж за период
   • Товары-аутсайдеры (что не продаётся)
   • Динамика продаж (сравнение периодов)
   • ABC-анализ товаров

СКЛАД И ЗАКУПКИ
   • Текущие остатки
   • Что заканчивается (дефицит)
   • Что залежалось (не продаётся 90+ дней)
   • Рекомендации по закупкам

ФИНАНСЫ
   • Выручка за период
   • Средний чек
   • Количество чеков

СВОБОДНЫЙ ВОПРОС
   • Спросите что угодно о вашей базе
   • Бот сам составит запрос и ответит


РЕШЕНИЕ ПРОБЛЕМ
---------------------------------------------------------------

"Python не найден"
   → Запустите INSTALL.bat от имени администратора

"Ошибка подключения к 1С"
   → Проверьте путь к базе в .env
   → Убедитесь что 1С установлена на этом компьютере
   → Попробуйте открыть базу обычным способом

"Ошибка Telegram"
   → Проверьте токен в .env (без лишних пробелов)
   → Убедитесь что интернет работает

"Ошибка LLM/OpenRouter"
   → Проверьте ключ PROVIDER_API_KEY
   → Зайдите на openrouter.ai и проверьте баланс
