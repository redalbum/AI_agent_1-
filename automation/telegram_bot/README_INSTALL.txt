   🤖 AI Бизнес-Ассистент для 1С — Инструкция по установке
═══════════════════════════════════════════════════════════════════════

ТРЕБОВАНИЯ:
   • Windows 10/11
   • Подключение к интернету (для установки)
   • Установленная 1С:Предприятие 8.x

УСТАНОВКА (6 простых шагов):

   1. Запустите INSTALL.bat от имени администратора
      (правый клик → "Запуск от имени администратора")
      
   2. Дождитесь завершения установки
   
   3. На рабочем столе появятся два ярлыка:
      • ⚙️ Настройки бота
      • 🚀 Запустить бота
      
   4. Откройте "⚙️ Настройки бота" и заполните:
      
      TELEGRAM_BOT_TOKEN
         Получите у @BotFather в Telegram:
         /newbot → введите имя → скопируйте токен
         
      TELEGRAM_ALLOWED_USERS  
         Ваш Telegram ID. Узнать: напишите @userinfobot
         
      ONEC_CONNECTION_STRING
         Путь к базе 1С, например:
         File="C:\Bases\МояБаза";
         
      PROVIDER_API_KEY
         Ключ от openrouter.ai:
         Зарегистрируйтесь → Keys → Create Key
         
   5. Сохраните файл и закройте
   
   6. Запустите "🚀 Запустить бота"

ИСПОЛЬЗОВАНИЕ:

   • Откройте Telegram
   • Найдите своего бота по имени
   • Напишите /start
   • Выберите команду из меню или задайте вопрос

ОСТАНОВКА:

   • Закройте окно бота, или
   • Нажмите Ctrl+C в окне бота, или  
   • Запустите STOP_BOT.bat

ПРОБЛЕМЫ И РЕШЕНИЯ:

   "Python не найден"
   → Перезагрузите компьютер после установки
   
   "Ошибка подключения к 1С"
   → Проверьте путь к базе в .env
   → Убедитесь что 1С не открыта в монопольном режиме
   
   "Ошибка Telegram"
   → Проверьте правильность токена
   → Убедитесь что указали свой Telegram ID
